# Electrical Engineering Prompt Library

本目录用于整理面向电气工程科技写作的 AI prompts，重点服务以下研究方向：

- 含分布式新能源配电网的故障特性分析
- 配电网保护与控制
- 配电网故障定位
- 配电网状态感知
- 构网型并网控制技术
- 构网型场站特性解析

## 目录说明

本 prompt 库聚焦以下任务：

1. 论文撰写
2. 英文润色
3. 中文转英文科技写作
4. 技术逻辑检查
5. 审稿与自审
6. 审稿意见回复（rebuttal）
7. 图表与实验分析写作
8. 专业术语、符号、单位一致性检查

## 文件规划

- `01_common_rules.md`：电气工程科技写作通用约束
- `02_writing_and_polishing.md`：论文撰写、翻译、润色 prompts
- `03_review_rebuttal.md`：审稿、修改、回复审稿意见 prompts

## 使用方式

每个 prompt 都可以直接复制到 ChatGPT、Claude、Gemini 或其他大模型中使用。  
建议在使用时补充以下材料：

- 论文段落或 LaTeX 片段
- 研究背景
- 研究对象与系统模型
- 仿真或实验结果
- 目标期刊或会议风格（如 IEEE）

## 当前适配重点

当前版本重点适配以下领域：

- 配电网故障分析
- 配电网保护与控制
- 故障定位
- 状态感知
- 构网型并网控制
- 构网型场站动态特性分析
